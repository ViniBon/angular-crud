Projeto executado com o intuito de desenvolver, adquirir e aprimorar todas e quaisquer habilidades relacionadas ao framework Angular, no qual foi desenvolvida uma página de cadastro com ações referentes às operações básicas do CRUD, utilizando técnicas de organização de pastas, rotas, serviços, contratos, etc.
  • Nomenclatura de pastas, arquivos, classes, funções, etc.
  • Organização de pastas e arquivos
  • Uso de bibliotecas externas
  • Uso de rotas
  • Uso de services para consumir APIs
  • Comunicação entre componentes (input e output, query params, route params)
  • Proatividade
  • Criatividade
  • Comportamento autodidata
  • Gestão de atividades
  • Angular
  • PO-UI
